# 🚀 XGBoost Model Deployment with Streamlit

Aplikasi web untuk prediksi menggunakan model XGBoost yang telah dilatih. Dibangun dengan Streamlit untuk kemudahan deployment dan interaksi.

## 📊 Tentang Proyek

Proyek ini mendeploy model **XGBoost Classifier** (4 kelas) dengan 19 fitur input. Model telah dilatih sebelumnya dan disimpan dalam format `.pkl`.

## ✨ Fitur

- Input untuk 19 fitur numerik
- Prediksi kelas secara real-time
- Menampilkan probabilitas per kelas
- Visualisasi bar chart
- Informasi model di sidebar

## 🛠️ Teknologi

- Python 3.9+
- Streamlit
- XGBoost
- Scikit-learn
- Pandas & NumPy
